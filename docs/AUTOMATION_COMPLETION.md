## 📋 API接口总览

### 系统状态接口
- `GET /api/automation?type=health` - 系统健康状态
- `GET /api/automation?type=dashboard` - 实时仪表板

### 用户分析接口
- `GET /api/automation?type=user-insights&userId={id}` - 用户洞察
- `GET /api/automation?type=analytics&userId={id}` - 用户行为分析
- `POST /api/automation?action=track-event` - 追踪用户事件

### 推荐系统接口
- `GET /api/automation?type=recommendations&userId={id}` - 获取推荐
- `POST /api/automation?action=recommendation-feedback` - 推荐反馈

### 营销自动化接口
- `POST /api/automation?action=create-campaign` - 创建营销活动
- `POST /api/automation?action=launch-campaign` - 启动营销活动
- `GET /api/automation?type=campaign-performance&campaignId={id}` - 活动性能

### 业务洞察接口
- `GET /api/automation?type=business-insights&insightType=growth` - 增长分析
- `GET /api/automation?type=business-insights&insightType=revenue` - 收入分析
- `GET /api/automation?type=business-insights&insightType=retention` - 留存分析
- `GET /api/automation?type=business-insights&insightType=ltv` - LTV分析
