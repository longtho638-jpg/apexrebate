## 🚀 立即部署指令

### 1. 环境准备
```bash
# 克隆项目
git clone <repository-url>
cd apexrebate

# 安装依赖
npm install

# 环境变量配置
cp .env.example .env.local
# 编辑 .env.local 配置生产环境变量
```

### 2. 数据库初始化
```bash
# 推送数据库架构
npx prisma db push

# 运行种子数据 (可选)
npm run seed
```

### 3. 构建和部署
```bash
# 构建项目
npm run build

# 启动生产服务
npm start
```
